# Decentralized Twitter DApp

A blockchain-based social media application built using **Solidity**. This project explores the fundamentals of smart contract development  enabling users to interact with a decentralized social feed directly on the Ethereum blockchain.

## 🚀 Overview
This DApp provides a platform for users to create and manage social posts in a transparent and immutable way. It serves as a practical implementation of core Solidity concepts for decentralized application (DApp) development.

## 🛠 Features
- **Smart Contract Logic:** Implements fundamental blockchain features including:
  - `Modifiers` for secure access control.
  - `Require` statements for input validation.
  - `Structs` to organize complex data (users, posts).
  - `Arrays` & `Mappings` for efficient data storage and retrieval.
  - `Inheritance` for clean, modular code architecture.
- **Development Environment:** Built and compiled using [Remix IDE](https://remix.ethereum.org/).
- **Live Testing:** Fully tested and verified on the **Sepolia Testnet**.

## ⚙️ How to Run
1. Open [Remix IDE](https://remix.ethereum.org/).
2. Create a new file and paste the `Twitter.sol` code.
3. Compile the contract using the Solidity compiler (version 0.8.x recommended).
4. Connect your MetaMask wallet, switch to the **Sepolia Testnet**, and deploy the contract using the "Injected Provider - MetaMask" environment.
5. Interact with the functions via the Remix dashboard.
